# Product Overview

## Node.js App

Angular SPA + Fastify REST API + MySQL によるフルスタック Web アプリケーション。

## ミッション

高い保守性とスケーラビリティを持ち、AI による継続的な修正に耐えうる堅牢なコードベースの構築。

## 主な機能

- JWT 認証（ログイン / ログアウト / 認証保護ルート）
- REST API（Fastify v4）
- Angular Material + Bootstrap 5 による UI
- MySQL 8 + Sequelize ORM によるデータ永続化
- Docker による開発環境の完全コンテナ化

## ユーザー体験の方針

- パフォーマンス・メンテナンス性・安全性・ユーザービリティを最優先
- OWASP Top 10 に準拠したセキュアな実装
